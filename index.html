Andrea Bocci Web Page#

Welcome to my web page 2!

I work in an innovation Lab as a Data Scientist, with the goal to bring new technologies and ideas into the business.

This site follows a similar approach: I experiment different layouts and documentation frameworks to write about data science (and other technical) topics that often includes code, plots or math equations.

Therefore this web site is an evolving experiment eternally work in progress, and an amalgam of different styles and layouts.

By design.

Why Sphinx?#

There are plenty of static web page framework out there, in the past I was using Pelican (that actually I recommend!) for example, but in recent years I was accumulating notes in notebook format and my wish-list basically included:

  • A pythonic framework

  • Write pages in markdown, better with richer capability (like reStructureText)

  • Easily convert notebooks to pages to be posted on this site

  • While having fun writing content to this web site, become more proficient on properly/automatically document code, projects and libraries.

Clearly there was nothing around that could fit all my requirements, in particular in the popular Jekyll ecosystem, as I still want to experiment with Jupyter Book, Quarto, Executable Books Project, Edward Tufte markdown and other stuff.

But eventually I found Sphinx with PyData-sphinx-theme a good compromise. Also I was convinced by this post from Chris Holdgraf, which gives quite a few compelling arguments on using it.

So, I did what any Data Scientist does to start from scratch a new project: I forked a repository!

In my case Chris Holdgraf web page, that I profoundly thank for the excellent work and documentation he makes his web site (still have to implement the GitHub Actions he uses to automatize the deployment).

Site Structure#

(Information taken from Chris Holdgraf web page)

  • Where is the blog source code? (TBD)

  • What engine builds the website? The Sphinx documentation engine is the core documentation engine.

  • What engine builds the blog? I use the ablog Sphinx blog extension to host the actual blog.

  • What theme do you use for the blog? I use the PyData Sphinx Theme

  • How do you write Sphinx docs in markdown? I use the myst-parser package

  • How do you write content in notebooks? I use the MyST-NB package.

  • How did you make those fancy buttons and dropdowns? That was the sphinx-panels Sphinx extension.
